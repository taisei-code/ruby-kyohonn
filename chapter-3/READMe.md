### 基数指示子を指定して、何進数で解釈させるか指定

2進数（**b**inary）

- 基数指示子
0b

- 例
0b10
# => 2

8進数（**O**ctal）

- 基数指示子
0oまたは0

- 例
0o10
# => 8

10進数（**d**ecimal）

- 基数指示子
0d

- 例
0d10
# => 10

16進数（he**x**adecimal）

- 基数指示子
0x

- 例
0x10
# => 16
